System Fonts are the Best Fonts

This page compares a simple text string set in Open Sans against the same text string set in your device’s system UI font. This is useful for seeing which weights of the system UI font are available on your device as well as determining whether switching from Open Sans to system fonts would result in text reflow.

— Matt Miklic
